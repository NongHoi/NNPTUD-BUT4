📋 Hướng dẫn Upload

🔐 Authentication

  • Cần JWT Token hợp lệ để upload
  • Token có thể lấy từ API đăng nhập
  • Token sẽ expire sau một thời gian
  • Kiểm tra định dạng: Bearer [token]

📸 Upload Avatar

  • Chỉ upload 1 file ảnh duy nhất
  • Kích thước tối đa: 5MB
  • Định dạng: JPG, PNG, GIF
  • Tỉ lệ khuyến nghị: 1:1 (vuông)

🖼️ Upload Multiple

  • Upload tối đa 10 files cùng lúc
  • Mỗi file tối đa 10MB
  • Hỗ trợ drag & drop
  • Preview trước khi upload

💡 Mẹo sử dụng

Bạn có thể kéo thả file trực tiếp vào vùng upload để tiết kiệm thời gian. Hệ thống sẽ tự động nhận diện và preview file trước khi upload.
